# Matrix-and-Tensor-Methods-in-Data-Analysis
This repository contains all assignments and tasks completed for the college course Matrične i tenzorske metode u analizi podataka (Matrix and Tensor Methods in Data Analysis). Each folder and file within this repository is organized according to the specific tasks and assignments undertaken during the course.

## Note
The original assignments were written in Croatian and translated into English for readability. While AI tools were used to assist with translation, all work presented here is the original work of my project team.
